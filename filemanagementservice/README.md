## Description

Hackathon File Management Service

## Installation

```bash
$ npm i
```

## Dependencies
- Consul - Service registry and discovery server
- MongoDB - NoSQL database

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```
